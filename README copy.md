# Student Learning Behavior Clustering Datasets

This package contains three independent dataset groups for the machine learning project:

1. Dataset 1 — Academic Performance
2. Dataset 2 — Behavioral / Lifestyle
3. Dataset 3 — Engagement / Activity

Important methodology note:
The original sources do not necessarily describe the same students. Therefore, the datasets were not force-merged by identity. Each dataset keeps a source/provenance column when applicable.

Recommended use:
- Use Dataset 1 for academic performance clustering.
- Use Dataset 2 for behavioral/lifestyle clustering.
- Use Dataset 3 as a lightweight supporting engagement/activity experiment.
- Compare clustering quality and interpretation across the three datasets.

Each subfolder contains:
- dataset files
- feature explanation markdown
- feature dictionary CSV
